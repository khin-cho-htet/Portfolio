# **App Name**: Khin Portfolio

## Core Features:

- Dynamic Mask Hero: A high-impact landing section featuring the profile photo with a geometric SVG mask and backdrop frame effects as requested.
- Experience Timeline: An interactive chronological journey through Senior BA and Junior Dev roles using Framer Motion for staggered reveals.
- AI Portfolio Chat Tool: An AI-powered assistant that uses reasoning to decide how to summarize Khin's experience for recruiters based on their specific hiring needs.
- Project Showcase Grid: A visual display of domain expertise including core banking, fintech systems, and linked digital platforms like Sarphat.
- Interactive Skills Cloud: A categorised view of professional skills from Agile to SDLC, optimized for readability and staggered animation.
- Certification Badge Gallery: A curated section for Google Project Management and Coursera credentials with verified link integrations.

## Style Guidelines:

- Primary color: Deep Amethyst (#9B86BD) conveying sophisticated professional wisdom and precision.
- Background color: Obsidian Blue (#12131A) to provide a premium, agency-style dark mode contrast.
- Accent color: Vibrant Orchid (#C57BCC) for active states, link hovers, and CTA borders.
- Headline font: 'Space Grotesk' for a computerized, tech-forward business analyst aesthetic; Body font: 'Inter' for neutral, machined readability.
- Clean, grid-based layout with significant whitespace and high-end digital agency margins.
- Complex staggered entry animations for text blocks and project cards, alongside a floating motion for the profile's background frame mask.
- Minimalist outline icons representing business metrics, code, and project management milestones.
